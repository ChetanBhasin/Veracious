Project: Veracious

By Chetan Bhasin and Anish George

Visual Big Data Mining web-application based on Apache Spark

Welcome to the Veracious project

Veracious is a web based application for mining massive datasets by combining datasets brought from different resources using a visual archecture.

Veracious is different from other data mining tools because it allows you to distribute the data processing over several nodes without touching any code.

We utilize the power of Apache Spark -- a framework written in Scala which uses a concept called Resilient Distributed Datasets (or RDDs) that are known for working really fast.

Apache Spark

Apache Spark is a top level Apache project created by Matei Zharia that aims to provide a simple alternative to Hadoop MapReduce framework to easily allow Big Data Engineers to create applications for solving problems pretaining to massive datastes.

Play Framework

Play Framework is a web-application framework for building Scala and Java applications. It is based on a lightweight, stateless, web-friendly architecture.

Built on Akka, Play provides predictable and minimal resource consumption (CPU, memory, threads) for highly-scalable applications.

Veracious is built on Play using the Scala language.

Akka toolkit

Akka is a toolkit and runtime or building highly concurrent, distributed, and resilient message-driven applications on the JVM.
